# Mishkam — Coming Soon Page

A modern luxury **coming soon** landing page for the Mishkam brand.

## Overview

Mishkam is a multi-vertical business group spanning fashion, food, retail, real estate, and services. This page serves as the pre-launch presence while the full platform is under development.

## Features

- Elegant dark luxury design with gold accent palette
- Animated staggered fade-in sections
- Responsive CSS grid for service cards with hover effects
- Google Fonts — Playfair Display, Cormorant Garamond, Jost
- Zero dependencies — pure HTML & CSS, no frameworks
